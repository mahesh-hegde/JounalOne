# JournalOne

A simple personal journal app written in flutter.

```py
JournalOne   [S][:]
------------------
[<][--Date----][>]
|----------------|
|                |
|                |
|                |
|                |
|--------------E-|
```

Options:
```
[ ] Dark theme
[ ] Export as JSON
[ ] Import JSON
```

Search screen
```
[search box]
[sort button]
[Matching entries in decreasing order, 3 lines each]
```
